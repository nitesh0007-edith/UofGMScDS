The Flow Concept

Transform a simple distribution (Gaussian) into a complex one through invertible functions.

x = f(z), where z ~ N(0, I)
🔑 Key Property

Because f is invertible, we can compute exact likelihoods using the change of variables formula:

log p(x) = log p(z) - log |det(∂f/∂z)|

Flow Architectures

  • RealNVP: Affine coupling layers
  • Glow: Invertible 1×1 convolutions
  • Neural Spline Flows: Flexible monotonic transformations

Key Takeaways

  • Flows: invertible transformations of simple distributions
  • Enable exact likelihood computation
  • Bidirectional: can generate and evaluate probability
  • Architecture must be carefully designed for tractable Jacobian
