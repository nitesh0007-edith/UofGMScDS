Reinforcement Learning

Learning from interaction—how agents learn to make decisions by maximizing cumulative rewards.

← PreviousNext →

The RL Framework

📖 Key Components

Agent: The learner/decision maker
Environment: What the agent interacts with
State: Current situation
Action: What agent can do
Reward: Feedback signal
Policy: Strategy for choosing actions

Markov Decision Processes

Formal framework: (S, A, P, R, γ)

G_t = R_{t+1} + γR_{t+2} + γ²R_{t+3} + ...

Goal: maximize expected cumulative discounted reward.

RL Methods

  • Value-based (Q-learning, DQN): Learn value of state-actions
  • Policy-based (REINFORCE): Directly optimize policy
  • Actor-Critic (A2C, PPO): Combine both approaches

Deep RL Breakthroughs

  • DQN (2015): Atari games from pixels
  • AlphaGo (2016): Defeated world Go champion
  • PPO: Stable, widely-used algorithm
  • RLHF: Training language models with human feedback

Key Takeaways

  • RL: learning through interaction and rewards
  • MDP provides formal framework
  • Value-based vs policy-based vs actor-critic
  • Deep RL: neural networks as function approximators
  • RLHF crucial for aligning language models
