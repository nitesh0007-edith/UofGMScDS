Why Does Deep Learning Work?

Theoretical insights into the surprising effectiveness of deep learning—double descent, lottery tickets, and implicit regularization.

← PreviousNext →

The Puzzles

Deep learning defies classical statistics:

  • Overparameterized models should overfit—but they generalize
  • Non-convex optimization should get stuck—but it finds good solutions
  • Networks can memorize random labels—but learn structure when present

Double Descent

Test error follows a "double descent" curve as model size increases:

🔑 The Phenomenon

1. Classical regime: error decreases then increases (overfitting)
2. Interpolation threshold: peak error
3. Overparameterized regime: error decreases again!

Lottery Ticket Hypothesis

Dense networks contain sparse "winning ticket" subnetworks that can achieve comparable performance when trained in isolation.

Implicit Regularization

SGD implicitly prefers simpler solutions—flat minima that generalize better.

Key Takeaways

  • Deep learning challenges classical ML theory
  • Double descent: bigger models can generalize better
  • Lottery tickets: sparse subnetworks exist within large models
  • SGD has implicit regularization effects
  • Theory is still catching up to practice
