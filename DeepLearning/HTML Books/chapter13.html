Graph Neural Networks

Learn to process graph-structured data—social networks, molecules, knowledge graphs—with message passing neural networks.

← Previous Next →

Why Graphs?

Many real-world data has graph structure:

  • Social networks: Users as nodes, friendships as edges
  • Molecules: Atoms as nodes, bonds as edges
  • Knowledge graphs: Entities and relationships
  • Traffic networks: Intersections and roads

Message Passing

The core GNN paradigm: nodes exchange information with neighbors.

h_v^(l+1) = UPDATE(h_v^(l), AGGREGATE({h_u^(l) : u ∈ N(v)}))
🔑 Message Passing Steps

1. Message: Each node creates message from its features
2. Aggregate: Collect messages from neighbors (sum, mean, max)
3. Update: Combine own features with aggregated messages
4. Repeat: Stack multiple layers for multi-hop information

Graph Learning Tasks

  • Node classification: Predict labels for nodes
  • Link prediction: Predict missing edges
  • Graph classification: Classify entire graphs (molecules)

Key Takeaways

  • GNNs process graph-structured data
  • Message passing: aggregate information from neighbors
  • Multiple layers = information from farther nodes
  • Tasks: node classification, link prediction, graph classification
  • Applications: drug discovery, social networks, recommendations
