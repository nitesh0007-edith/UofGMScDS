🌦️ Glasgow Weather Visualization

Multi-view Interactive Analysis of Daily Weather Data (2015-2019)

Information Visualisation Group Project | University of Glasgow | 2024/25

Explore Glasgow's Weather Patterns

This project presents three distinct visualization systems for analyzing 1,795 days of Glasgow weather data. Each system uses different design approaches to support five analytical tasks: seasonal comparison, extreme event identification, correlation exploration, temporal trend analysis, and interactive filtering.

System A

Temporal Analysis & Correlation

Traditional charts optimized for trend detection and pattern exploration with bidirectional linking.

  • Time series line chart (temperature trends)
  • Scatter plot with regression (humidity vs visibility)
  • Seasonal comparison bar chart
  • Wind speed distribution histogram
  • Bidirectional brushing & linking
Launch System A
System B

Statistical Distribution Focus

Distribution-focused design emphasizing outlier detection and pattern recognition through statistical views.

  • Monthly temperature heatmap (year × month)
  • Box plots showing quartiles & outliers
  • Scatter plot with dynamic regression
  • Weather type distribution bars
  • Click & brush selection across views
Launch System B
System C

Small Multiples & Faceting

Maximizes pattern visibility through faceted views and explicit filtering controls for detailed exploration.

  • Faceted time series (one per year)
  • Strip plot for outlier identification
  • Multi-dimensional bubble chart
  • Temperature distribution histogram
  • Dropdown & slider filtering controls
Launch System C
System A + Generalization

Hierarchical Temporal Selection

Extended System A with generalized selection demonstrating semantic hierarchical abstraction.

  • 5-level temporal hierarchy (Day→Year)
  • Interactive hierarchy traversal
  • Aggregation at all levels (week/month/season/year)
  • Visual feedback on selection generalization
  • NOT global filtering - true semantic abstraction
