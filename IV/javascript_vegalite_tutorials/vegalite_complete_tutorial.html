Part 1: Introduction to Vega-Lite

What is Vega-Lite?

Vega-Lite is a high-level grammar of interactive graphics. It provides a concise JSON syntax for creating visualizations that are:

  • Declarative: Describe what you want, not how to draw it
  • Data-driven: Visualizations are built from data
  • Interactive: Built-in support for user interactions
  • Web-based: Runs in any modern browser

Example 1.1: Your First Vega-Lite Chart

A simple scatter plot showing the relationship between horsepower and fuel efficiency.

// Basic Vega-Lite specification structure const spec1_1 = { "$schema": "https://vega.github.io/schema/vega-lite/v5.json", "description": "A simple scatter plot", "data": { "url": "https://vega.github.io/vega-lite/data/cars.json" }, "mark": "point", // Visual element type "encoding": { // Map data to visual properties "x": {"field": "Horsepower", "type": "quantitative"}, "y": {"field": "Miles_per_Gallon", "type": "quantitative"} } };

Example 1.2: Enhanced Chart with Multiple Encodings

Adding color, size, and tooltips to show more information.

// Enhanced chart with multiple encoding channels const spec1_2 = { "$schema": "https://vega.github.io/schema/vega-lite/v5.json", "data": {"url": "https://vega.github.io/vega-lite/data/cars.json"}, "mark": "point", "encoding": { "x": {"field": "Horsepower", "type": "quantitative"}, "y": {"field": "Miles_per_Gallon", "type": "quantitative"}, "color": {"field": "Origin", "type": "nominal"}, // Color by origin "size": {"field": "Weight_in_lbs", "type": "quantitative"}, // Size by weight "tooltip": [ // Show multiple fields on hover {"field": "Name", "type": "nominal"}, {"field": "Horsepower", "type": "quantitative"}, {"field": "Miles_per_Gallon", "type": "quantitative"} ] }, "width": 600, "height": 400, "title": "Car Performance: Horsepower vs MPG" };

Part 2: Marks and Encoding

Marks are the visual elements that represent data:

  • point - Scatter plot points
  • bar - Bar charts
  • line - Line charts
  • area - Filled areas
  • rect - Rectangles (heatmaps)
  • text - Text labels

Encoding channels map data to visual properties:

  • x, y - Position
  • color - Color
  • size - Size
  • shape - Shape
  • opacity - Transparency

Data types affect interpretation:

  • quantitative - Continuous numbers
  • nominal - Unordered categories
  • ordinal - Ordered categories
  • temporal - Dates/times

Example 2.1: Bar Chart

Count of cars by origin.

const spec2_1 = { "$schema": "https://vega.github.io/schema/vega-lite/v5.json", "data": {"url": "https://vega.github.io/vega-lite/data/cars.json"}, "mark": "bar", // Bar mark type "encoding": { "x": {"field": "Origin", "type": "nominal"}, "y": {"aggregate": "count", "type": "quantitative"}, // Count records "color": {"field": "Origin", "type": "nominal"} }, "width": 400, "title": "Number of Cars by Origin" };

Example 2.2: Line Chart

Average MPG over time.

const spec2_2 = { "$schema": "https://vega.github.io/schema/vega-lite/v5.json", "data": {"url": "https://vega.github.io/vega-lite/data/cars.json"}, "mark": {"type": "line", "point": true}, // Line with points "encoding": { "x": {"field": "Year", "type": "ordinal", "title": "Year"}, "y": { "aggregate": "mean", // Calculate average "field": "Miles_per_Gallon", "type": "quantitative", "title": "Average MPG" }, "tooltip": [ {"field": "Year", "type": "ordinal"}, {"aggregate": "mean", "field": "Miles_per_Gallon", "type": "quantitative"} ] }, "width": 600, "title": "Average MPG Over Time" };

Example 2.3: Comparing Data Types

Cylinders encoded as quantitative (continuous) vs ordinal (discrete).

// Two charts side by side showing different data type encodings const spec2_3 = { "$schema": "https://vega.github.io/schema/vega-lite/v5.json", "data": {"url": "https://vega.github.io/vega-lite/data/cars.json"}, "hconcat": [ // Horizontal concatenation { "mark": "point", "encoding": { "x": {"field": "Cylinders", "type": "quantitative"}, // Continuous "y": {"field": "Miles_per_Gallon", "type": "quantitative"}, "color": {"field": "Cylinders", "type": "quantitative"} }, "width": 250, "title": "Cylinders as Quantitative" }, { "mark": "point", "encoding": { "x": {"field": "Cylinders", "type": "ordinal"}, // Discrete "y": {"field": "Miles_per_Gallon", "type": "quantitative"}, "color": {"field": "Cylinders", "type": "ordinal"} }, "width": 250, "title": "Cylinders as Ordinal" } ] };

Part 3: Data Transformation

Transform data before visualization without modifying the source:

  • Filter: Select subset of data
  • Calculate: Create new fields
  • Aggregate: Summarize data (mean, sum, count, etc.)
  • Bin: Group continuous data into bins
  • Window: Running calculations

Example 3.1: Filtering Data

Show only fuel-efficient cars (MPG > 30).

const spec3_1 = { "$schema": "https://vega.github.io/schema/vega-lite/v5.json", "data": {"url": "https://vega.github.io/vega-lite/data/cars.json"}, "transform": [ {"filter": "datum.Miles_per_Gallon > 30"} // Filter condition ], "mark": "point", "encoding": { "x": {"field": "Horsepower", "type": "quantitative"}, "y": {"field": "Miles_per_Gallon", "type": "quantitative"}, "color": {"field": "Origin", "type": "nominal"}, "tooltip": [ {"field": "Name", "type": "nominal"}, {"field": "Miles_per_Gallon", "type": "quantitative"} ] }, "width": 600, "title": "Fuel-Efficient Cars (MPG > 30)" };

Example 3.2: Calculate New Fields

Create horsepower per cylinder metric.

const spec3_2 = { "$schema": "https://vega.github.io/schema/vega-lite/v5.json", "data": {"url": "https://vega.github.io/vega-lite/data/cars.json"}, "transform": [ { "calculate": "datum.Horsepower / datum.Cylinders", // New field "as": "HP_per_Cylinder" } ], "mark": "point", "encoding": { "x": {"field": "Cylinders", "type": "ordinal"}, "y": {"field": "HP_per_Cylinder", "type": "quantitative"}, "color": {"field": "Origin", "type": "nominal"}, "tooltip": [ {"field": "Name", "type": "nominal"}, {"field": "HP_per_Cylinder", "type": "quantitative", "format": ".2f"} ] }, "width": 600, "title": "Horsepower per Cylinder" };

Example 3.3: Binning (Histogram)

Distribution of horsepower using bins.

const spec3_3 = { "$schema": "https://vega.github.io/schema/vega-lite/v5.json", "data": {"url": "https://vega.github.io/vega-lite/data/cars.json"}, "mark": "bar", "encoding": { "x": { "bin": {"maxbins": 20}, // Create bins "field": "Horsepower", "type": "quantitative" }, "y": { "aggregate": "count", "type": "quantitative" }, "tooltip": [ {"bin": true, "field": "Horsepower"}, {"aggregate": "count"} ] }, "width": 600, "title": "Distribution of Horsepower" };

Part 4: Scales, Axes, and Legends

Scales map data values to visual values.

Axes provide reference marks for position scales.

Legends provide reference marks for color, size, shape scales.

All can be customized for better communication.

Example 4.1: Custom Color Scale

Define specific colors for each origin.

const spec4_1 = { "$schema": "https://vega.github.io/schema/vega-lite/v5.json", "data": {"url": "https://vega.github.io/vega-lite/data/cars.json"}, "mark": {"type": "point", "filled": true, "size": 100}, "encoding": { "x": {"field": "Horsepower", "type": "quantitative"}, "y": {"field": "Miles_per_Gallon", "type": "quantitative"}, "color": { "field": "Origin", "type": "nominal", "scale": { "domain": ["USA", "Europe", "Japan"], // Define categories "range": ["#e74c3c", "#3498db", "#2ecc71"] // Define colors }, "legend": { "title": "Country of Origin", "orient": "right", "titleFontSize": 14, "labelFontSize": 12 } }, "tooltip": [ {"field": "Name", "type": "nominal"}, {"field": "Origin", "type": "nominal"} ] }, "width": 600, "height": 400, "title": "Custom Color Scale" };

Example 4.2: Customizing Axes

Format axis titles, labels, and grid.

const spec4_2 = { "$schema": "https://vega.github.io/schema/vega-lite/v5.json", "data": {"url": "https://vega.github.io/vega-lite/data/cars.json"}, "mark": "point", "encoding": { "x": { "field": "Horsepower", "type": "quantitative", "axis": { "title": "Horsepower (HP)", "titleFontSize": 14, "titleFontWeight": "bold", "grid": true, "gridColor": "lightgray" } }, "y": { "field": "Miles_per_Gallon", "type": "quantitative", "axis": { "title": "Fuel Efficiency (MPG)", "titleFontSize": 14, "titleFontWeight": "bold", "format": ".1f", // Format numbers "grid": true } }, "color": {"field": "Origin", "type": "nominal"} }, "width": 600, "height": 400, "title": "Customized Axes" };

Example 4.3: Logarithmic Scale

Use log scale for better visualization of skewed data.

const spec4_3 = { "$schema": "https://vega.github.io/schema/vega-lite/v5.json", "data": {"url": "https://vega.github.io/vega-lite/data/cars.json"}, "mark": "point", "encoding": { "x": { "field": "Horsepower", "type": "quantitative", "scale": {"type": "log"}, // Logarithmic scale "axis": {"title": "Horsepower (log scale)"} }, "y": { "field": "Miles_per_Gallon", "type": "quantitative", "scale": {"zero": false} // Don't force zero baseline }, "color": {"field": "Origin", "type": "nominal"} }, "width": 600, "title": "Logarithmic Scale Example" };

Part 5: Multi-view Composition

Combine multiple views to create complex visualizations:

  • hconcat - Horizontal concatenation (side by side)
  • vconcat - Vertical concatenation (stacked)
  • layer - Overlay charts on same axes
  • facet - Create small multiples
  • repeat - Repeat charts with different fields

Example 5.1: Dashboard Layout

Multiple views arranged in a dashboard.

const spec5_1 = { "$schema": "https://vega.github.io/schema/vega-lite/v5.json", "data": {"url": "https://vega.github.io/vega-lite/data/cars.json"}, "vconcat": [ // Vertical concatenation { "hconcat": [ // Horizontal concatenation { "mark": "point", "encoding": { "x": {"field": "Horsepower", "type": "quantitative"}, "y": {"field": "Miles_per_Gallon", "type": "quantitative"}, "color": {"field": "Origin", "type": "nominal"} }, "width": 300, "height": 250, "title": "HP vs MPG" }, { "mark": "bar", "encoding": { "x": {"bin": {"maxbins": 15}, "field": "Horsepower"}, "y": {"aggregate": "count"} }, "width": 300, "height": 250, "title": "HP Distribution" } ] }, { "mark": "bar", "encoding": { "x": {"field": "Origin", "type": "nominal"}, "y": {"aggregate": "mean", "field": "Miles_per_Gallon"}, "color": {"field": "Origin", "type": "nominal"} }, "width": 620, "title": "Average MPG by Origin" } ] };

Example 5.2: Layered Chart

Scatter plot with regression line.

const spec5_2 = { "$schema": "https://vega.github.io/schema/vega-lite/v5.json", "data": {"url": "https://vega.github.io/vega-lite/data/cars.json"}, "layer": [ // Layer multiple marks { "mark": "point", "encoding": { "x": {"field": "Horsepower", "type": "quantitative"}, "y": {"field": "Miles_per_Gallon", "type": "quantitative"}, "color": {"field": "Origin", "type": "nominal"} } }, { "mark": {"type": "line", "color": "red", "size": 3}, "transform": [ { "regression": "Miles_per_Gallon", // Calculate regression "on": "Horsepower" } ], "encoding": { "x": {"field": "Horsepower", "type": "quantitative"}, "y": {"field": "Miles_per_Gallon", "type": "quantitative"} } } ], "width": 600, "height": 400, "title": "Scatter Plot with Regression Line" };

Example 5.3: Faceted Chart (Small Multiples)

Separate charts for each category.

const spec5_3 = { "$schema": "https://vega.github.io/schema/vega-lite/v5.json", "data": {"url": "https://vega.github.io/vega-lite/data/cars.json"}, "mark": "point", "encoding": { "x": {"field": "Horsepower", "type": "quantitative"}, "y": {"field": "Miles_per_Gallon", "type": "quantitative"}, "color": {"field": "Origin", "type": "nominal"}, "column": {"field": "Origin", "type": "nominal"} // Facet by origin }, "width": 250, "height": 250, "title": "HP vs MPG by Origin (Faceted)" };

Part 6: Interaction

Make visualizations interactive and explorable:

  • Selection: Click or drag to select data
  • Binding: Connect to UI elements (dropdowns, sliders)
  • Conditional Encoding: Change appearance based on selection
  • Linked Views: Selection affects multiple charts
  • Tooltips: Show details on hover

Example 6.1: Click to Highlight

Click on legend to filter data.

const spec6_1 = { "$schema": "https://vega.github.io/schema/vega-lite/v5.json", "data": {"url": "https://vega.github.io/vega-lite/data/cars.json"}, "params": [{ "name": "origin_selection", "select": {"type": "point", "fields": ["Origin"]}, // Selection parameter "bind": "legend" // Bind to legend }], "mark": "point", "encoding": { "x": {"field": "Horsepower", "type": "quantitative"}, "y": {"field": "Miles_per_Gallon", "type": "quantitative"}, "color": { "field": "Origin", "type": "nominal", "condition": { "param": "origin_selection", // Conditional encoding "value": null } }, "opacity": { "condition": {"param": "origin_selection", "value": 1}, "value": 0.2 }, "size": { "condition": {"param": "origin_selection", "value": 100}, "value": 30 } }, "width": 600, "height": 400, "title": "Click on Legend to Filter" };

Example 6.2: Brush Selection

Drag to select a region.

const spec6_2 = { "$schema": "https://vega.github.io/schema/vega-lite/v5.json", "data": {"url": "https://vega.github.io/vega-lite/data/cars.json"}, "params": [{ "name": "brush", "select": {"type": "interval"} // Interval selection (brush) }], "mark": "point", "encoding": { "x": {"field": "Horsepower", "type": "quantitative"}, "y": {"field": "Miles_per_Gallon", "type": "quantitative"}, "color": { "condition": { "param": "brush", "field": "Origin", "type": "nominal" }, "value": "lightgray" }, "size": { "condition": {"param": "brush", "value": 100}, "value": 30 } }, "width": 600, "height": 400, "title": "Drag to Select Region (Brush)" };

Example 6.3: Linked Views

Selection in one chart affects another.

const spec6_3 = { "$schema": "https://vega.github.io/schema/vega-lite/v5.json", "data": {"url": "https://vega.github.io/vega-lite/data/cars.json"}, "params": [{ "name": "brush_linked", "select": {"type": "interval"} // Shared selection }], "hconcat": [ { "mark": "point", "encoding": { "x": {"field": "Horsepower", "type": "quantitative"}, "y": {"field": "Miles_per_Gallon", "type": "quantitative"}, "color": { "condition": { "param": "brush_linked", "field": "Origin", "type": "nominal" }, "value": "lightgray" } }, "width": 300, "height": 300, "title": "Select Here" }, { "mark": "point", "encoding": { "x": {"field": "Weight_in_lbs", "type": "quantitative"}, "y": {"field": "Acceleration", "type": "quantitative"}, "color": { "condition": { "param": "brush_linked", "field": "Origin", "type": "nominal" }, "value": "lightgray" } }, "width": 300, "height": 300, "title": "Linked View" } ] };

Example 6.4: Dropdown Filter

Use dropdown menu to filter by origin.

const spec6_4 = { "$schema": "https://vega.github.io/schema/vega-lite/v5.json", "data": {"url": "https://vega.github.io/vega-lite/data/cars.json"}, "params": [{ "name": "origin_select", "select": {"type": "point", "fields": ["Origin"]}, "bind": { "input": "select", "options": [null, "USA", "Europe", "Japan"], // Dropdown options "labels": ["All", "USA", "Europe", "Japan"], "name": "Origin: " }, "value": null }], "mark": "point", "encoding": { "x": {"field": "Horsepower", "type": "quantitative"}, "y": {"field": "Miles_per_Gallon", "type": "quantitative"}, "color": {"field": "Origin", "type": "nominal"}, "opacity": { "condition": {"param": "origin_select", "value": 1}, "value": 0.1 } }, "width": 600, "height": 400, "title": "Filter by Origin Using Dropdown" };

Example 6.5: Slider Filter

Use slider to filter by year.

const spec6_5 = { "$schema": "https://vega.github.io/schema/vega-lite/v5.json", "data": {"url": "https://vega.github.io/vega-lite/data/cars.json"}, "params": [{ "name": "year_slider", "value": 1970, "bind": { "input": "range", // Slider input "min": 1970, "max": 1982, "step": 1, "name": "Minimum Year: " } }], "transform": [ {"filter": "datum.Year >= year_slider"} // Filter based on slider ], "mark": "point", "encoding": { "x": {"field": "Horsepower", "type": "quantitative"}, "y": {"field": "Miles_per_Gallon", "type": "quantitative"}, "color": {"field": "Origin", "type": "nominal"}, "tooltip": [ {"field": "Name", "type": "nominal"}, {"field": "Year", "type": "ordinal"} ] }, "width": 600, "height": 400, "title": "Filter by Year Using Slider" };
