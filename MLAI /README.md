## Assessed Coursework

### Coursework 1 — ECG Explainable Machine Learning
**Folder:** `ECG_Explainable_Machine_Learning/`  
Group project using the MIT-BIH arrhythmia dataset to classify ECG heartbeat types and explain model predictions using **LIME** and **SHAP**.

### Coursework 3 — PCA on 3D Shape Analysis
**Folder:** `PCA_in_3D_Shape_Analysis/`  
Applied Principal Component Analysis to 3D lumbar spine mesh sequences to identify dominant modes of shape variation.

## Topics Covered

- Regression (Parts 1–3) and regularisation
- Classification: Naive Bayes, logistic regression, SVMs (primal & dual)
- Unsupervised learning: K-means, GMMs, EM algorithm
- Dimensionality reduction: PCA, kernel PCA
- Model explainability: LIME, SHAP
- Constrained optimisation
