Java Basics - Foundation Concepts 📚

1. Primitive Types

Java has 8 primitive types. For exams, you'll mainly use:

Type Size Example Usage in Exams
int 32-bit int count = 10; Counts, prices, IDs
double 64-bit double price = 19.99; Rarely used in exams
boolean 1-bit boolean available = true; Flags, availability

3. Arrays

Fixed-size collections of elements

Java 
// Create array - Method 1
int[] numbers = new int[5];  // [0, 0, 0, 0, 0]

// Create array - Method 2
String[] names = {"Alice", "Bob", "Charlie"};

// Access elements (0-indexed!)
String first = names[0];  // "Alice"

// Array length
int size = names.length;  // 3 (no parentheses!)

// Loop through array - Method 1
for (int i = 0; i < names.length; i++) {
    System.out.println(names[i]);
}

// Loop through array - Method 2 (enhanced for-loop)
for (String name : names) {
    System.out.println(name);
}

⚠️ Common Array Mistakes

  • Index out of bounds: Remember arrays are 0-indexed!
  • Length vs length(): Arrays use .length, Strings use .length()
  • Fixed size: Once created, array size cannot change (use ArrayList for dynamic size)

Classes & Objects - The Heart of Java 🏗️

What is a Class?

A class is a blueprint for creating objects. Think of it like a cookie cutter - the class is the cutter, and objects are the cookies!

Class vs Object

Class: Animal
(Blueprint)
creates
Object: dog1
(Buddy)
Object: dog2
(Max)
Object: cat1
(Whiskers)

1. Basic Class Structure

Java 
package petcare;  // Always declare package first!

import java.util.Objects;  // For equals/hashCode

public class Animal {
    // Fields (instance variables) - PRIVATE!
    private String name;
    private int age;

    // Constructor - same name as class
    public Animal(String name, int age) {
        this.name = name;  // 'this' refers to current object
        this.age = age;
    }

    // Getter methods - PUBLIC!
    public String getName() {
        return name;
    }

    public int getAge() {
        return age;
    }

    // Setter method - with validation!
    public void setName(String name) {
        if (name == null || name.length() < 3) {
            throw new IllegalArgumentException("Name too short");
        }
        this.name = name;
    }
}

✅ Encapsulation Rule

Fields should be PRIVATE, methods should be PUBLIC

This is called encapsulation - hiding the internal details!

2. Constructors with Validation

Constructors initialize objects. Always validate in constructors!

Java - Constructor Validation Pattern 
public Animal(String name, int age) {
    // Validate BEFORE assigning!

    // Check name not null and min length
    if (name == null || name.length() < 3) {
        throw new IllegalArgumentException("Name must be at least 3 characters");
    }

    // Check age is positive
    if (age < 0) {
        throw new IllegalArgumentException("Age must be positive");
    }

    // Only assign if validation passes
    this.name = name;
    this.age = age;
}

3. The Three Essential Override Methods

These methods are inherited from Object class. You'll override them in almost every exam!

Java - toString() 
// toString() - returns String representation
@Override
public String toString() {
    return name + " (age " + age + ")";
}

// Usage:
Animal dog = new Animal("Buddy", 5);
System.out.println(dog);  // Calls toString() automatically!
// Output: Buddy (age 5)
Java - equals() and hashCode() 
// equals() - compares two objects
@Override
public boolean equals(Object obj) {
    // Step 1: Check if same object
    if (this == obj) return true;

    // Step 2: Check if null or different class
    if (obj == null || getClass() != obj.getClass()) return false;

    // Step 3: Cast and compare fields
    Animal other = (Animal) obj;
    return age == other.age && Objects.equals(name, other.name);
}

// hashCode() - MUST override with equals!
@Override
public int hashCode() {
    return Objects.hash(name, age);
}

⚠️ When to Override equals() and hashCode()?

Always override when:

  • Objects will be stored in a Set
  • Objects will be used as keys in a Map
  • You need to compare objects by their content (not reference)

Rule: If you override equals(), you MUST override hashCode() too!

public enum SeatType {
    STANDARD, DELUXE;

    // Helper method to convert from string
    public static SeatType fromString(String s) {

        if (s.equals("D")) return DELUXE;

for (String part : parts) {

📝 Exam Pattern: Size Comparison

You'll often need to compare enum sizes (e.g., can a SMALL animal fit in a LARGE enclosure?)

Java - Size Compatibility 
public boolean checkCompatibility(Animal animal) {
    // LARGE animal needs LARGE enclosure
    if (animal.getSize() == AnimalSize.LARGE && this.size != AnimalSize.LARGE) {
        return false;
    }

    // MEDIUM animal needs MEDIUM or LARGE enclosure
    if (animal.getSize() == AnimalSize.MEDIUM && this.size == AnimalSize.SMALL) {
        return false;
    }

    // SMALL animal fits anywhere
    return true;
}

Collections Framework - Dynamic Data Structures 📦

Why Collections?

Arrays are fixed size. Collections are dynamic - they grow and shrink as needed!

Three Main Collection Types

📝 List
(Ordered, Allows Duplicates)

Use when: Order matters, need indexing

🎯 Set
(Unique, No Order)

Use when: No duplicates allowed

🗂️ Map
(Key-Value Pairs)

Use when: Need lookup by key

1. List - ArrayList

Most common collection in exams!

Java - ArrayList Basics 
import java.util.ArrayList;
import java.util.List;

// Create ArrayList
List<String> names = new ArrayList<>();

// Add elements
names.add("Alice");
names.add("Bob");
names.add("Charlie");

// Get element by index
String first = names.get(0);  // "Alice"

// Size (note: it's a method!)
int count = names.size();  // 3

// Check if contains
boolean hasBob = names.contains("Bob");  // true

// Remove element
names.remove("Bob");

// Loop through - Method 1
for (int i = 0; i < names.size(); i++) {
    System.out.println(names.get(i));
}

// Loop through - Method 2 (preferred!)
for (String name : names) {
    System.out.println(name);
}

2. Set - HashSet

No duplicates allowed! Perfect for unique collections.

Java - HashSet Basics 
import java.util.HashSet;
import java.util.Set;

// Create HashSet
Set<String> uniqueNames = new HashSet<>();

// Add elements
uniqueNames.add("Alice");
uniqueNames.add("Bob");
uniqueNames.add("Alice");  // Won't be added - duplicate!

// Size
int count = uniqueNames.size();  // 2 (not 3!)

// Check if contains
boolean hasAlice = uniqueNames.contains("Alice");  // true

// Loop through (no guaranteed order!)
for (String name : uniqueNames) {
    System.out.println(name);
}

✅ Why Set Needs equals() and hashCode()

Set uses hashCode() to quickly find where objects should go, and equals() to check if they're duplicates.

Without these methods overridden, Set won't work correctly with your custom objects!

3. Map - HashMap

Store key-value pairs. Perfect for inventories!

Java - HashMap Basics 
import java.util.HashMap;
import java.util.Map;

// Create HashMap
Map<String, Integer> ages = new HashMap<>();

// Put key-value pairs
ages.put("Alice", 25);
ages.put("Bob", 30);
ages.put("Charlie", 28);

// Get value by key
int aliceAge = ages.get("Alice");  // 25

// Check if key exists
boolean hasBob = ages.containsKey("Bob");  // true

// Get with default value (SUPER USEFUL!)
int davidAge = ages.getOrDefault("David", 0);  // 0 (not in map)

// Update existing value
ages.put("Alice", 26);  // Overwrites old value

📝 Exam Pattern: Inventory with Map

Used in trading systems to track goods!

Java - Inventory Pattern 
public class Citizen {
    private int gems;
    private Map<Goods, Integer> inventory;

    public Citizen(int gems) {
        this.gems = gems;
        this.inventory = new HashMap<>();
    }

    // Get amount of specific goods
    public int getAmount(Goods goods) {
        return inventory.getOrDefault(goods, 0);
    }

    // Add goods to inventory
    public void addGoods(Goods goods, int amount) {
        int current = getAmount(goods);
        inventory.put(goods, current + amount);
    }

    // Execute trade
    public boolean executeTrade(Trade trade) {
        if (gems < trade.getGems()) {
            return false;  // Not enough gems!
        }

        gems -= trade.getGems();
        addGoods(trade.getGoods(), trade.getAmount());
        return true;
    }
}

Pattern 1: Data Class with Validation

Examples: Animal, Seat, Node, GameCharacter, Trade

Marks: 6-8 marks

Time: 12-15 minutes

Java - Template 
package packagename;

import java.util.Objects;

public class DataClass {
    // Step 1: Private fields
    private String field1;
    private int field2;

    // Step 2: Constructor with validation
    public DataClass(String field1, int field2) {
        // Validate field1
        if (field1 == null || field1.length() < 3) {
            throw new IllegalArgumentException("Field1 too short");
        }
        // Validate field2
        if (field2 < 0 || field2 > 100) {
            throw new IllegalArgumentException("Field2 out of range");
        }
        this.field1 = field1;
        this.field2 = field2;
    }

    // Step 3: Getters
    public String getField1() { return field1; }
    public int getField2() { return field2; }

    // Step 4: Setters (with validation!)
    public void setField1(String field1) {
        if (field1 == null || field1.length() < 3) {
            throw new IllegalArgumentException("Field1 too short");
        }
        this.field1 = field1;
    }

    // Step 5: toString()
    @Override
    public String toString() {
        return field1 + " (" + field2 + ")";
    }

    // Step 6: equals() and hashCode()
    @Override
    public boolean equals(Object obj) {
        if (this == obj) return true;
        if (obj == null || getClass() != obj.getClass()) return false;
        DataClass other = (DataClass) obj;
        return field2 == other.field2 && Objects.equals(field1, other.field1);
    }

    @Override
    public int hashCode() {
        return Objects.hash(field1, field2);
    }
}

Pattern 2: String Parser

Examples: GraphParser, Venue constructor

Marks: 4-8 marks

Time: 10-15 minutes

Java - Parser Template 
public static Type[] parseFromString(String spec) {
    // Step 1: Split by newlines
    String[] lines = spec.split("\n");

    // Step 2: Get count from first line
    int count = Integer.valueOf(lines[0]);

    // Step 3: Create array/collection
    Type[] result = new Type[count];

    // Step 4: Parse each line
    for (int i = 0; i < count; i++) {
        String[] parts = lines[i + 1].split(" ");

        // Step 5: Extract data and create objects
        int value1 = Integer.valueOf(parts[0]);
        String value2 = parts[1];

        result[i] = new Type(value1, value2);
    }

    return result;
}

✅ String Parsing Checklist

  • ✅ Split by newline: split("\n")
  • ✅ Split by space: split(" ")
  • ✅ Convert to int: Integer.valueOf(str)
  • ✅ Remember: arrays are 0-indexed!

Pattern 3: Manager/Container Class

Examples: PetService, Venue, Player, Trader

Marks: 8-15 marks

Time: 15-20 minutes

Java - Manager Template 
public class Manager {
    // Step 1: Collection to manage items
    private List<Item> items;

    // Step 2: Constructor initializes empty collection
    public Manager() {
        this.items = new ArrayList<>();
    }

    // Step 3: Add item method
    public void addItem(Item item) {
        items.add(item);
    }

    // Step 4: Print/display method
    public void printAll() {
        for (Item item : items) {
            System.out.println(item);
        }
    }

    // Step 5: Find best match method
    public Item findBest(/* criteria */) {
        Item best = null;
        int bestValue = Integer.MAX_VALUE;

        for (Item item : items) {
            if (/* matches criteria */) {
                if (item.getValue() < bestValue) {
                    bestValue = item.getValue();
                    best = item;
                }
            }
        }

        return best;
    }

    // Step 6: Remove item method
    public void removeItem(Item item) {
        if (!items.contains(item)) {
            throw new IllegalArgumentException("Item not found");
        }
        items.remove(item);
    }
}

package library;

import java.util.Objects;

public class Book {
    // TODO: Add private fields
    private String isbn;           // Must be exactly 13 characters
    private String title;          // Min 1 character
    private BookCategory category;
    private boolean isAvailable;

    // TODO: Constructor with validation
    public Book(String isbn, String title, BookCategory category) {
        // Validate ISBN length = 13
        if (isbn == null || isbn.length() != 13) {
            throw new IllegalArgumentException("ISBN must be 13 characters");
        }

        // Validate title not empty
        if (title == null || title.length() < 1) {
            throw new IllegalArgumentException("Title cannot be empty");
        }

        this.isbn = isbn;
        this.title = title;
        this.category = category;
        this.isAvailable = true;  // Initially available
    }

    // TODO: Getters
    public String getIsbn() { return isbn; }
    public String getTitle() { return title; }
    public BookCategory getCategory() { return category; }
    public boolean isAvailable() { return isAvailable; }

    // TODO: Setter for availability only
    public void setAvailable(boolean available) {
        this.isAvailable = available;
    }

    // TODO: Override toString()
    @Override
    public String toString() {
        return title + " (ISBN: " + isbn + ", " + category + ", " +
               (isAvailable ? "Available" : "Borrowed") + ")";
    }

    // TODO: Override equals() - based on ISBN only
    @Override
    public boolean equals(Object obj) {
        if (this == obj) return true;
        if (obj == null || getClass() != obj.getClass()) return false;
        Book book = (Book) obj;
        return Objects.equals(isbn, book.isbn);
    }

    // TODO: Override hashCode()
    @Override
    public int hashCode() {
        return Objects.hash(isbn);
    }
}

📝 Skeleton Code - Shelf Class

Java - Shelf.java (15-18 mins) 
package library;

import java.util.ArrayList;
import java.util.List;

public class Shelf {
    // TODO: Add private fields
    private int capacity;           // Max books this shelf can hold
    private String location;        // e.g., "A1", "B2"
    private BookCategory category;  // Shelf holds only one category
    private List<Book> books;

    // TODO: Constructor
    public Shelf(int capacity, String location, BookCategory category) {
        this.capacity = capacity;
        this.location = location;
        this.category = category;
        this.books = new ArrayList<>();
    }

    // TODO: Getters
    public int getCapacity() { return capacity; }
    public String getLocation() { return location; }
    public BookCategory getCategory() { return category; }
    public List<Book> getBooks() { return books; }

    // TODO: Check if book can fit on this shelf
    public boolean canAddBook(Book book) {
        // Check 1: Shelf not full
        if (books.size() >= capacity) {
            return false;
        }

        // Check 2: Book category matches shelf category
        if (book.getCategory() != this.category) {
            return false;
        }

        return true;
    }

    // TODO: Add book to shelf
    public void addBook(Book book) {
        if (!canAddBook(book)) {
            throw new IllegalArgumentException("Cannot add book to this shelf");
        }
        books.add(book);
    }

    // TODO: Remove book from shelf
    public void removeBook(Book book) {
        if (!books.contains(book)) {
            throw new IllegalArgumentException("Book not on this shelf");
        }
        books.remove(book);
    }

    // TODO: Check if shelf has space
    public boolean hasSpace() {
        return books.size() < capacity;
    }
}

📝 Skeleton Code - Library Class

Java - Library.java (15-18 mins) 
package library;

import java.util.ArrayList;
import java.util.List;

public class Library {
    // TODO: Add private field
    private List<Shelf> shelves;

    // TODO: Constructor
    public Library() {
        this.shelves = new ArrayList<>();
    }

    // TODO: Add shelf to library
    public void addShelf(Shelf shelf) {
        shelves.add(shelf);
    }

    // TODO: Print all shelves and their books
    public void printAllShelves() {
        for (int i = 0; i < shelves.size(); i++) {
            Shelf shelf = shelves.get(i);
            System.out.println("Shelf " + shelf.getLocation() +
                             " (" + shelf.getCategory() + ") - " +
                             shelf.getBooks().size() + "/" + shelf.getCapacity() + " books");

            for (Book book : shelf.getBooks()) {
                System.out.println("  - " + book);
            }
        }
    }

    // TODO: Add book to best available shelf
    public boolean addBook(Book book) {
        // Find first shelf that can hold this book
        for (Shelf shelf : shelves) {
            if (shelf.canAddBook(book)) {
                shelf.addBook(book);
                return true;
            }
        }
        return false;  // No shelf available
    }

    // TODO: Find and remove a book
    public void removeBook(Book book) {
        for (Shelf shelf : shelves) {
            if (shelf.getBooks().contains(book)) {
                shelf.removeBook(book);
                return;
            }
        }
        throw new IllegalArgumentException("Book not found in library");
    }

    // TODO: Find book by ISBN
    public Book findBookByISBN(String isbn) {
        for (Shelf shelf : shelves) {
            for (Book book : shelf.getBooks()) {
                if (book.getIsbn().equals(isbn)) {
                    return book;
                }
            }
        }
        return null;
    }
}

Common Mistakes & How to Avoid Them

Mistake Why It Happens Solution
Missing package declaration Forget to add at top of file Write package in first file, copy-paste to all others
Array index out of bounds Forgetting arrays are 0-indexed Always use i < array.length, not i <= array.length
NullPointerException Not checking for null Always check: if (obj == null) before using obj
.length vs .length() Confusion between arrays and strings Arrays: .length (no parentheses), Strings: .length()
Set doesn't work correctly Forgot to override equals/hashCode Always override both for objects in Sets/Maps
Validation only in constructor Forgetting about setters Copy validation logic to setter methods
