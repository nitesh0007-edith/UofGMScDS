## Examination Details

**Course:** Introduction to Data Science and Systems (COMPSCI 5089)
**Duration:** 2 hours
**Total Marks:** 80 marks
**Type:** Open book, online assessment
**Questions:** Answer all 4 questions

## Solution Coverage

### Question 1: Linear Algebra (20 marks)
- (a) Adjacency matrix for parcel shipment graph
- (b) Steady state calculations using eigenvalue and power iteration methods
- (c) SVD decomposition and pseudo-inverse calculation

**Topics:** Graph theory, transition matrices, SVD, Moore-Penrose inverse

### Question 2: Optimization (20 marks)
- (a) Least squares using normal equations
- (b) Gradient descent with 2 iterations
- (c) Stochastic gradient descent (SGD) merits for large datasets
- (d) Constrained optimization using Lagrange multipliers

**Topics:** Linear least squares, gradient descent, SGD, constrained optimization

### Question 3: Probabilities (20 marks)
- (a) Normal distribution parametrization and MLE estimation
- (b) Gaussian Mixture Models (GMM) as alternative to single Gaussian
- (c) Bayesian inference: P(Buy | gaze location) using Bayes' theorem

**Topics:** Multivariate normal, GMM, EM algorithm, Bayesian classification

### Question 4: Databases (20 marks)
- (a) Block calculations, join cardinality, selectivity in query optimization
- (b) Database selection for music streaming service with ML embeddings
- (c) Vector database indexing strategies (HNSW, IVF)

**Topics:** Database design, vector databases, indexing, similarity search
