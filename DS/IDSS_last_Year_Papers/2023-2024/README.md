## Examination Details

**Date:** December 2023
**Course:** Introduction to Data Science and Systems (COMPSCI 5089)
**Duration:** Standard exam duration
**Total Marks:** 80 marks
**Format:** Open book, online assessment
**Questions:** Answer all 4 questions

## Solution Coverage

### Question 1: Linear Algebra (20 marks)
- (a) **Data whitening** - Complete step-by-step process with all matrix dimensions
- (b) **Effect on height/weight variables** - Why whitening is desirable with examples
- (c) **Matrix transformation from point correspondences** - Calculate 2×2 matrix M from two points, overdetermined systems with three points

**Topics:** Whitening, covariance matrices, eigendecomposition, linear transformations, least squares

### Question 2: Optimization (20 marks)
- (a) **Contour plot analysis** - Characterize loss function at three specific points, gradient direction and magnitude
- (b) **Algorithm identification** - Identify gradient descent, hill-climbing, and Newton's method from trajectories
- (c) **Local minima issues** - Problems with iterative algorithms and mitigation strategies
- (d) **Constrained optimization** - Avoid specific local minimum using constraints

**Topics:** Gradient descent, Newton's method, local vs global optima, constrained optimization, simulated annealing

### Question 3: Probability and Statistics (20 marks)
- (a) **Expectation calculations** - Exam, coursework, and weighted final mark expectations
- (b) **Conditional probabilities** - Pass rates overall and by group, statistical analysis
- (c) **Weight optimization** - Approach to set assessment weightings for target pass rate

**Topics:** Expectation, conditional probability, optimization for target outcomes, cross-validation

### Question 4: Database Systems (20 marks)
- (a) **Blocking factors** - Calculate records per block for Course and Transcript relations
- (b) **Storage requirements** - Number of blocks needed for each relation
- (c) **Join cost analysis** - Nested loop join with two strategies, compare efficiency
- (d) **Selection cardinality** - Concept and significance in query optimization

**Topics:** Physical database design, blocking factors, join algorithms, query optimization
