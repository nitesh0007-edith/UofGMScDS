## How to Create PDF from HTML

The HTML file contains beautifully formatted solutions with proper styling. To convert it to PDF:

### Method 1: Using Your Web Browser (Recommended)
1. Open `IDSS_2019-2020_Solutions.html` in your web browser (Chrome, Safari, Firefox, or Edge)
2. Press `Cmd+P` (Mac) or `Ctrl+P` (Windows/Linux) to open Print dialog
3. Select "Save as PDF" as the destination
4. Adjust settings if needed:
   - Paper size: A4 or Letter
   - Margins: Default or Custom
   - Options: Enable "Background graphics" for better appearance
5. Click "Save" and name your PDF file

### Method 2: Using macOS Preview
1. Open `IDSS_2019-2020_Solutions.html` in Safari
2. Go to File → Export as PDF
3. Save the PDF file

### Method 3: Using Command Line (if you install wkhtmltopdf)
```bash
brew install wkhtmltopdf
wkhtmltopdf IDSS_2019-2020_Solutions.html IDSS_2019-2020_Solutions.pdf
```

- (a) Cosine similarity calculations, applications, TF-IDF implementation, Lemmatization vs Stemming

### Question 3: Database Systems (20 marks)
- (a) Blocking factor calculations, Query processing costs for different file organizations
- (b) Nested-loop join strategies with cost analysis, Index-based join optimization
