🎯 What is Probability?

Probability is the language we use to talk about uncertainty. In everyday life, we constantly deal with situations where we don't know exactly what will happen:

  • Will it rain tomorrow?
  • Will I roll a 6 on this die?
  • Where is my lost phone?

Probability gives us a way to measure how likely different outcomes are, using numbers between 0 and 1:

Probability Scale

0.0 ─────── 0.5 ─────── 1.0
Impossible Even Odds Certain

🌟 Real-World Example: Finding a Lost Submarine

Imagine a submarine is lost somewhere in the ocean. We divide the ocean into a grid of squares. The submarine is definitely in one of these squares, but we don't know which one.

Key Questions:

  • How likely is the submarine in square [5, 7]?
  • Which squares should we search first?
  • How do we update our beliefs after searching some squares?

This is exactly what probability theory helps us answer!

🎲 Random Variables

Definition: Random Variable

A random variable is a variable whose value is determined by chance. We use capital letters like \(X\), \(Y\), or \(Z\) to denote random variables.

Types of Random Variables

Type Description Examples
Discrete Can only take specific, countable values Rolling a die: {1, 2, 3, 4, 5, 6}
Number of students in a class
Continuous Can take any value in a range Temperature: 20.5°C, 20.51°C, ...
Height of a person

In this guide, we focus on discrete random variables.

🎲 Example: Rolling a Fair Die

Let \(X\) be the outcome of rolling a fair six-sided die.

  • Sample Space: \(X \in \{1, 2, 3, 4, 5, 6\}\)
  • Each outcome is equally likely: \(P(X=1) = P(X=2) = \cdots = P(X=6) = \frac{1}{6}\)

📊 Probability Mass Function (PMF)

Definition: Probability Mass Function

A Probability Mass Function (PMF) tells us the probability that a discrete random variable equals each specific value.

\[f_X(x) = P(X = x)\]

This reads as: "The PMF of \(X\) evaluated at \(x\) equals the probability that \(X\) equals \(x\)"

Visualizing a PMF

Think of a PMF as a map that shows how "probability mass" is distributed across all possible outcomes. Higher values mean that outcome is more likely.

✅ Properties of Valid PMFs

For a function to be a valid PMF, it must satisfy TWO essential properties:

Property 1: Non-negativity

All probabilities must be between 0 and 1:

\[0 \leq P(X = x) \leq 1 \quad \text{for all } x\]

Why? Probability of 0 means impossible, 1 means certain. Nothing can be "more than certain" or "less than impossible"!

Property 2: Normalization

All probabilities must sum to 1:

\[\sum_{\text{all } x} P(X = x) = 1\]

Why? Something must happen! One of the possible outcomes must occur, so the total probability is 100% = 1.0

🎯 Example: Checking if a PMF is Valid

Example 1 - Valid PMF:

For a random variable \(X \in \{A, B, C\}\):

  • \(P(X = A) = 0.2\)
  • \(P(X = B) = 0.3\)
  • \(P(X = C) = 0.5\)

✓ All values between 0 and 1
✓ Sum = 0.2 + 0.3 + 0.5 = 1.0
This IS a valid PMF!

Example 2 - Invalid PMF:

For a random variable \(Y \in \{1, 2, 3\}\):

  • \(P(Y = 1) = 0.4\)
  • \(P(Y = 2) = 0.5\)
  • \(P(Y = 3) = 0.3\)

✓ All values between 0 and 1
✗ Sum = 0.4 + 0.5 + 0.3 = 1.2 ≠ 1.0
This is NOT valid! Probabilities exceed 100%

🧮 Computing Probabilities

Single Outcome

To find the probability of a single outcome, simply look it up in the PMF:

\[P(X = 3) = f_X(3)\]

Multiple Outcomes (Events)

An event is a set of outcomes. To find the probability of an event, sum the probabilities of all outcomes in that event.

\[P(X \in \{a, b, c\}) = P(X=a) + P(X=b) + P(X=c)\]

🎲 Example: Probability of Rolling an Even Number

Let \(X\) be the outcome of a fair die roll.

Event: Rolling an even number = \(\{2, 4, 6\}\)

\[\begin{align} P(\text{even}) &= P(X=2) + P(X=4) + P(X=6)\\ &= \frac{1}{6} + \frac{1}{6} + \frac{1}{6}\\ &= \frac{3}{6} = \frac{1}{2} = 0.5 \end{align}\]

So there's a 50% chance of rolling an even number!

Complementary Probability

The probability of an event NOT happening is 1 minus the probability it happens:

\[P(\text{NOT } A) = 1 - P(A)\]

🎲 Example: Probability of NOT Rolling a 6

\[\begin{align} P(X \neq 6) &= 1 - P(X = 6)\\ &= 1 - \frac{1}{6}\\ &= \frac{5}{6} \approx 0.833 \end{align}\]

📈 Expected Value

Definition: Expected Value

The expected value (or mean) is the average outcome if we repeated the random process many, many times.

\[E[X] = \sum_{\text{all } x} x \cdot P(X = x)\]

In words: Multiply each possible value by its probability, then sum everything up.

🎲 Example: Expected Value of a Die Roll

\[\begin{align} E[X] &= 1 \cdot P(X=1) + 2 \cdot P(X=2) + \cdots + 6 \cdot P(X=6)\\ &= 1 \cdot \frac{1}{6} + 2 \cdot \frac{1}{6} + 3 \cdot \frac{1}{6} + 4 \cdot \frac{1}{6} + 5 \cdot \frac{1}{6} + 6 \cdot \frac{1}{6}\\ &= \frac{1 + 2 + 3 + 4 + 5 + 6}{6}\\ &= \frac{21}{6} = 3.5 \end{align}\]

Important! The expected value doesn't have to be a possible outcome. You can't roll 3.5, but on average, you get 3.5 per roll.

💡 Key Insight

For a function of a random variable, the expected value is:

\[E[f(X)] = \sum_{\text{all } x} f(x) \cdot P(X = x)\]

Warning: In general, \(E[f(X)] \neq f(E[X])\)

You must apply the function FIRST, then compute the expectation!

🎯 Example: Expected Value of \(X^2\) for a Die Roll

Method 1 (WRONG): \(E[X]^2\)

\[E[X]^2 = (3.5)^2 = 12.25 \quad \text{❌ Incorrect!}\]

Method 2 (CORRECT): \(E[X^2]\)

\[\begin{align} E[X^2] &= 1^2 \cdot \frac{1}{6} + 2^2 \cdot \frac{1}{6} + 3^2 \cdot \frac{1}{6} + 4^2 \cdot \frac{1}{6} + 5^2 \cdot \frac{1}{6} + 6^2 \cdot \frac{1}{6}\\ &= \frac{1 + 4 + 9 + 16 + 25 + 36}{6}\\ &= \frac{91}{6} \approx 15.17 \quad \text{✓ Correct!} \end{align}\]

🌍 Real-World Examples

🎰 Example 1: Lottery Ticket

You buy a lottery ticket for $1. There are three possible outcomes:

  • Win $100 with probability 0.001 (0.1%)
  • Win $10 with probability 0.01 (1%)
  • Win nothing with probability 0.989 (98.9%)

Is this a valid PMF?

\[0.001 + 0.01 + 0.989 = 1.0 \quad \text{✓ Yes!}\]

What's the expected winnings?

\[\begin{align} E[\text{Winnings}] &= 100 \cdot 0.001 + 10 \cdot 0.01 + 0 \cdot 0.989\\ &= 0.10 + 0.10 + 0\\ &= \$0.20 \end{align}\]

On average, you win 20 cents per ticket. Since tickets cost $1, you lose 80 cents on average! 📉

👨‍👩‍👧 Example 2: Family Size

In a certain town, the distribution of number of children per family is:

Number of Children Probability
0 0.30
1 0.25
2 0.25
3 0.15
4 or more 0.05

Questions:

1. What's the probability a randomly selected family has at least 2 children?

\[\begin{align} P(X \geq 2) &= P(X=2) + P(X=3) + P(X \geq 4)\\ &= 0.25 + 0.15 + 0.05\\ &= 0.45 = 45\% \end{align}\]

2. What's the expected number of children per family (treating "4 or more" as exactly 4)?

\[\begin{align} E[X] &= 0 \cdot 0.30 + 1 \cdot 0.25 + 2 \cdot 0.25 + 3 \cdot 0.15 + 4 \cdot 0.05\\ &= 0 + 0.25 + 0.50 + 0.45 + 0.20\\ &= 1.40 \text{ children} \end{align}\]

🎲 Example 3: Rolling Two Dice

When you roll two dice and add them, what's the PMF of the sum?

The sum can range from 2 (rolling 1+1) to 12 (rolling 6+6). But not all sums are equally likely!

Sum Ways to Get It Probability
2 (1,1) 1/36
3 (1,2), (2,1) 2/36
4 (1,3), (2,2), (3,1) 3/36
5 (1,4), (2,3), (3,2), (4,1) 4/36
6 (1,5), (2,4), (3,3), (4,2), (5,1) 5/36
7 (1,6), (2,5), (3,4), (4,3), (5,2), (6,1) 6/36

...and so on (symmetric pattern continues for 8-12)

Most likely sum: 7 with probability 6/36 = 1/6 ≈ 16.7%

🎓 Summary: Key Takeaways

  • PMF: Maps each possible value to its probability
  • Valid PMF: All probabilities in [0,1] and sum to 1
  • Events: Sum probabilities of outcomes in the event
  • Expected Value: Weighted average using probabilities
  • Important: \(E[f(X)] \neq f(E[X])\) in general
