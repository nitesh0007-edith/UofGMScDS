🎯 Introduction: What is Information?

Imagine you're searching for a lost submarine. You have two scenarios:

Scenario 1: Submarine equally likely in any of 256 squares
Uncertainty: VERY HIGH

Scenario 2: You know it's in one specific square
Uncertainty: ZERO

Information Theory gives us mathematical tools to:

  • 📊 Quantify uncertainty (entropy)
  • 📈 Measure how much we learn from observations
  • 🎯 Decide which search locations give us the most information
  • 🔍 Compare different probability distributions

📊 Entropy: Measuring Uncertainty

Definition: Shannon Entropy

Entropy measures the average uncertainty or "surprise" in a random variable.

\[H(X) = -\sum_{x} P(x) \log_2 P(x)\]

Units: Bits (when using log base 2)

Interpretation: Average number of yes/no questions needed to determine the outcome

Intuitive Understanding

Think of entropy as answering: "How surprised would I be on average?"

🎲 Example 1: Fair Coin

Coin flip: P(Heads) = 0.5, P(Tails) = 0.5

\[\begin{align} H(X) &= -P(H) \log_2 P(H) - P(T) \log_2 P(T)\\ &= -0.5 \log_2(0.5) - 0.5 \log_2(0.5)\\ &= -0.5 \times (-1) - 0.5 \times (-1)\\ &= 0.5 + 0.5 = 1 \text{ bit} \end{align}\]

Interpretation: You need exactly 1 yes/no question to determine the outcome.

🎲 Example 2: Unfair Coin

Biased coin: P(Heads) = 0.9, P(Tails) = 0.1

\[\begin{align} H(X) &= -0.9 \log_2(0.9) - 0.1 \log_2(0.1)\\ &= -0.9 \times (-0.152) - 0.1 \times (-3.322)\\ &= 0.137 + 0.332\\ &= 0.469 \text{ bits} \end{align}\]

Much lower entropy! You're rarely surprised - it's almost always heads.

🎲 Example 3: Certain Outcome

Always heads: P(Heads) = 1.0, P(Tails) = 0.0

\[\begin{align} H(X) &= -1.0 \log_2(1.0) - 0.0 \log_2(0.0)\\ &= -1.0 \times 0 - 0\\ &= 0 \text{ bits} \end{align}\]

Zero entropy! No uncertainty, never surprised.

Note: We define \(0 \log 0 = 0\) by convention (limit as p→0)

🎲 Example 4: Fair Die

Six-sided die: P(each face) = 1/6

\[\begin{align} H(X) &= -\sum_{i=1}^{6} \frac{1}{6} \log_2\left(\frac{1}{6}\right)\\ &= -6 \times \frac{1}{6} \times \log_2\left(\frac{1}{6}\right)\\ &= -\log_2\left(\frac{1}{6}\right)\\ &= \log_2(6) \approx 2.585 \text{ bits} \end{align}\]

You need about 2.585 yes/no questions on average to determine the outcome.

✅ Properties of Entropy

Key Properties

1. Non-negativity:

\[H(X) \geq 0\]

Entropy is always non-negative. Zero only when outcome is certain.

2. Maximum Entropy:

\[H(X) \leq \log_2(n)\]

For n possible outcomes, entropy is maximized when all outcomes are equally likely (uniform distribution).

3. Additivity for Independent Variables:

\[H(X, Y) = H(X) + H(Y) \quad \text{if X and Y are independent}\]
Entropy Spectrum

0 bits ←──────────────────→ Maximum
Certain Uncertain Uniform

Examples with 4 outcomes:
[1.0, 0, 0, 0] → H = 0 bits (certain)
[0.7, 0.2, 0.05, 0.05] → H = 1.19 bits
[0.4, 0.3, 0.2, 0.1] → H = 1.85 bits
[0.25, 0.25, 0.25, 0.25] → H = 2 bits (maximum)

Conditional Entropy

Conditional Entropy

The average uncertainty remaining about X after observing Y:

\[H(X \mid Y) = \sum_{y} P(y) H(X \mid Y=y)\]

Expanded form:

\[H(X \mid Y) = -\sum_{x,y} P(x,y) \log_2 P(x \mid y)\]

🔑 Important Relationship

\[H(X, Y) = H(X) + H(Y \mid X) = H(Y) + H(X \mid Y)\]

Chain Rule for Entropy!

📏 KL Divergence

Definition: Kullback-Leibler Divergence

KL divergence measures how different two probability distributions are.

\[D_{KL}(P \| Q) = \sum_{x} P(x) \log_2 \frac{P(x)}{Q(x)}\]

Interpretation: Expected extra bits needed when using distribution Q to encode data actually from distribution P.

⚠️ Important Properties

  • Non-negative: \(D_{KL}(P \| Q) \geq 0\)
  • Zero iff identical: \(D_{KL}(P \| Q) = 0\) only when P = Q
  • NOT symmetric: \(D_{KL}(P \| Q) \neq D_{KL}(Q \| P)\) in general
  • NOT a metric: Doesn't satisfy triangle inequality

📊 Example: Comparing Distributions

True distribution P: [0.5, 0.3, 0.2]

Approximate distribution Q: [0.4, 0.4, 0.2]

\[\begin{align} D_{KL}(P \| Q) &= 0.5 \log_2\frac{0.5}{0.4} + 0.3 \log_2\frac{0.3}{0.4} + 0.2 \log_2\frac{0.2}{0.2}\\ &= 0.5 \times 0.322 + 0.3 \times (-0.415) + 0.2 \times 0\\ &= 0.161 - 0.125 + 0\\ &= 0.036 \text{ bits} \end{align}\]

The distributions are quite similar (low KL divergence).

🔗 Mutual Information

Definition: Mutual Information

Mutual information measures how much knowing one variable tells you about another.

\[I(X; Y) = H(X) - H(X \mid Y) = H(Y) - H(Y \mid X)\]

Alternative formula:

\[I(X; Y) = \sum_{x,y} P(x,y) \log_2 \frac{P(x,y)}{P(x)P(y)}\]

Interpretation: Reduction in uncertainty about X after observing Y.

🔑 Key Insights

  • Symmetric: \(I(X; Y) = I(Y; X)\)
  • Non-negative: \(I(X; Y) \geq 0\)
  • If independent: \(I(X; Y) = 0\)
  • Maximum: \(I(X; Y) = H(X) = H(Y)\) when X and Y are deterministically related

🔍 Example: Submarine Search

Before search:

Submarine equally likely in 256 squares

\[H(\text{Location}) = \log_2(256) = 8 \text{ bits}\]

After searching one square and not finding it:

Now 255 remaining squares (approximately)

\[H(\text{Location | Not found}) \approx \log_2(255) \approx 7.99 \text{ bits}\]

Information gained:

\[I = 8 - 7.99 = 0.01 \text{ bits}\]

Small information gain! This guides search strategy - search where you expect to find it!

🌍 Real-World Applications

🎯 Application 1: Decision Trees

When building decision trees, choose splits that maximize information gain:

\[\text{Information Gain} = H(\text{Before Split}) - H(\text{After Split})\]

Example: Predicting if someone plays tennis

Initial entropy (50% yes, 50% no): H = 1 bit

After split on "Weather":

  • Sunny: 80% no → H = 0.72 bits
  • Rainy: 80% yes → H = 0.72 bits

Information Gain = 1.0 - 0.72 = 0.28 bits

📱 Application 2: Data Compression

Entropy gives a lower bound on how much we can compress data:

\[\text{Minimum bits per symbol} = H(X)\]

English text: H ≈ 1.5 bits per character

ASCII uses 8 bits, so theoretically we could compress to ~19% of original size!

🔬 Application 3: Feature Selection

Choose features with highest mutual information with the target:

\[I(\text{Feature}; \text{Target})\]

High mutual information = feature is informative for prediction

🎲 Application 4: Twenty Questions Game

Optimal strategy: Ask questions that split probability space in half!

With 1024 possible items:

\[H = \log_2(1024) = 10 \text{ bits}\]

You need 10 yes/no questions if you choose optimally!

🎓 Summary: Key Takeaways

Entropy:

\[H(X) = -\sum_{x} P(x) \log_2 P(x)\]
  • Measures average uncertainty
  • Maximum for uniform distribution
  • Zero for certain outcomes

KL Divergence:

\[D_{KL}(P \| Q) = \sum_{x} P(x) \log_2 \frac{P(x)}{Q(x)}\]
  • Measures difference between distributions
  • Non-negative, zero iff P = Q
  • NOT symmetric

Mutual Information:

\[I(X; Y) = H(X) - H(X \mid Y)\]
  • Measures shared information
  • Symmetric
  • Zero for independent variables
