1️⃣ Axioms of Probability OFFICIAL

📌 Boundedness

\[0 \leq P(A) \leq 1\]
What it means: All probabilities are between 0 (impossible) and 1 (certain).
🎯 Exam use: Check if given probability is valid. If P(X) = 1.2 or P(Y) = -0.3, it's INVALID!

📌 Unitarity (Normalization)

\[\sum_x P(x) = 1\]
What it means: All possible outcomes must sum to 100%.
🎯 Exam use:
  • Verify PMF is valid
  • Find missing probability: P(C) = 1 - P(A) - P(B)
  • Check your answers sum to 1

📌 Sum Rule

\[P(A \vee B) = P(A) + P(B) - P(A \wedge B)\]
What it means: Probability of "A OR B" = subtract overlap to avoid double-counting.
🎯 Exam use: When asked "at least one event happens". If A and B are mutually exclusive, P(A∧B)=0, so just add.

2️⃣ Conditional Probability OFFICIAL

\[P(A|B) = \frac{P(A \wedge B)}{P(B)} = \frac{P(A,B)}{P(B)}\]
What it means: Probability of A given B happened. Restricts sample space to only cases where B is true.
🎯 Exam use:
  • Problems with "given that..."
  • Update probabilities with new info
  • Rearrange to find P(A,B) = P(A|B)·P(B)
Example: P(Rain|Cloudy) = P(Rain,Cloudy)/P(Cloudy)
⚠️ Common mistake: P(A|B) ≠ P(B|A) in general!

3️⃣ Marginal Probability OFFICIAL

📌 Discrete

\[P(X) = \sum_y P(X,Y)\]

📌 Continuous

\[P(X) = \int_y P(X,Y) \, dy\]
What it means: "Sum out" or "integrate out" other variables to get probability of X alone.
🎯 Exam use:
  • Given joint P(X,Y) table, sum rows/cols for marginals
  • Convert joint distribution to single variable
  • Use with Law of Total Probability
💡 Tip: In joint probability tables, marginals are the row/column totals!

4️⃣ Bayes' Rule OFFICIAL

\[P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)}\]
What it means: Flip conditional probabilities! Go from P(B|A) to P(A|B).
🎯 Exam use:
  • Medical test: P(Disease|Positive) from P(Positive|Disease)
  • Spam filter: P(Spam|Word) from P(Word|Spam)
  • Diagnostic: Update beliefs with evidence
Expanded form when P(B) unknown: \[P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B|A) \cdot P(A) + P(B|\neg A) \cdot P(\neg A)}\]
Pattern: Know P(Test+|Sick), want P(Sick|Test+) → Use Bayes Rule!

5️⃣ Integration to Unity OFFICIAL

📌 Discrete

\[\sum_i f_X(x_i) = 1\]

📌 Continuous

\[\int_{-\infty}^{\infty} f_X(x) \, dx = 1\]
What it means: PDF/PMF must integrate/sum to 1 (total probability = 100%).
🎯 Exam use:
  • Find normalizing constant in PDF
  • Verify if function is valid PDF/PMF
  • Find unknown parameter c in f(x) = c·g(x)
If f(x) = c·x² on [0,1], find c: ∫₀¹ c·x² dx = 1 → c·[x³/3]₀¹ = 1 → c = 3

6️⃣ Expectation OFFICIAL

📌 Discrete

\[E[X] = \sum_x x \cdot f_X(x)\]

📌 Continuous

\[E[X] = \int_{-\infty}^{\infty} x \cdot f_X(x) \, dx\]
What it means: Average value if experiment repeated infinitely. "Center of mass" of distribution.
🎯 Exam use:
  • Calculate mean/average
  • Find expected payoff/cost/value
  • Use E[X] = μ for normal distributions

📌 Expectation of Function g(X)

\[E[g(X)] = \sum_x f_X(x) \cdot g(x) \quad \text{(discrete)}\]
\[E[g(X)] = \int_{-\infty}^{\infty} f_X(x) \cdot g(x) \, dx \quad \text{(continuous)}\]
⚠️ Critical: E[g(X)] ≠ g(E[X]) in general! Must apply g FIRST, then take expectation.
E[X²] ≠ (E[X])² Need E[X²] for variance: Var(X) = E[X²] - (E[X])²
Linearity: E[aX + b] = aE[X] + b
Sum: E[X + Y] = E[X] + E[Y] (always!)

7️⃣ Odds and Logit OFFICIAL

📌 Odds

\[\text{odds} = \frac{p}{1-p}\]
What it means: Ratio of "happens" to "doesn't happen". Alternative to probability.
🎯 Exam use:
  • Convert probability ↔ odds
  • Interpret betting odds
  • Logistic regression problems

📌 Logit (Log-Odds)

\[\text{logit}(p) = \log\left(\frac{p}{1-p}\right)\]
What it means: Log of odds. Maps [0,1] to (-∞,+∞).
🎯 Exam use: Logistic regression link function. If logit(p) = 0, then p = 0.5.

8️⃣ Cumulative Distribution OFFICIAL

\[F_X(x) = \int_{-\infty}^x f_X(t) \, dt = P(X \leq x)\]
What it means: Probability that X is at most x. Accumulates probability from -∞ to x.
🎯 Exam use:
  • P(X ≤ a) = F(a)
  • P(X > a) = 1 - F(a)
  • P(a < X ≤ b) = F(b) - F(a)
  • Recover PDF: f(x) = dF/dx
Properties:
  • F(-∞) = 0
  • F(+∞) = 1
  • Non-decreasing
  • Right-continuous

9️⃣ Normal Distribution OFFICIAL

\[f_X(x) = \frac{1}{Z} e^{-\frac{(x-\mu)^2}{2\sigma^2}}\]
where Z = σ√(2π)
What it means: Bell curve distribution. Characterized by mean μ (center) and variance σ² (spread).
🎯 Exam use:
  • Recognize normal: X ~ N(μ, σ²)
  • 68% within μ ± σ
  • 95% within μ ± 2σ
  • 99.7% within μ ± 3σ
  • Standardize: Z = (X-μ)/σ ~ N(0,1)
💡 Quick check: If asked for P(X < μ), answer is 0.5 (symmetry!)
IQ ~ N(100, 15²) P(85 < IQ < 115) ≈ 68% (within 1 std dev)

🔟 Mean and Variance OFFICIAL

📌 Sample Mean

\[\hat{\mu} = \frac{1}{N} \sum_{i=1}^N x_i\]
What it means: Average of your data. Hat (^) indicates it's an estimate.
🎯 Exam use:
  • Calculate from data: add all, divide by N
  • Estimates population mean μ
  • Use in MLE for normal distribution

📌 Sample Variance

\[\hat{\sigma}^2 = \frac{1}{N} \sum_{i=1}^N (x_i - \hat{\mu})^2\]
What it means: Average squared deviation from mean. Measures spread.
🎯 Exam use:
  • Step 1: Calculate μ̂
  • Step 2: Find (xᵢ - μ̂)² for each point
  • Step 3: Average these squared deviations
  • Standard deviation: σ̂ = √(σ̂²)
⚠️ Note: Divide by N (biased) or N-1 (unbiased). Check problem statement!
Alternative formula: Var(X) = E[X²] - (E[X])²
Easier when you already know E[X] and E[X²]

📝 EXAM STRATEGY GUIDE

🎯 Problem Type Recognition:
  • "Given that..." → Conditional probability P(A|B)
  • "Find probability of..." → Use PMF/PDF or CDF
  • "Reverse the condition" → Bayes' Rule
  • "Average" or "expected" → Expectation E[X]
  • "Spread" or "variability" → Variance
  • "At least one" → Sum rule or complement
  • "Both happen" → Joint probability P(A,B)
✓ Always check:
  • Does probability sum/integrate to 1?
  • Are all probabilities in [0,1]?
  • Did I use the right formula?
  • Do units make sense?

🔄 Common Exam Patterns

Pattern 1: Medical/Test Diagnosis

Given: P(Test+|Disease), P(Disease) Find: P(Disease|Test+) → Use Bayes' Rule!

Pattern 2: Joint → Marginal

Given: P(X,Y) table Find: P(X) or P(Y) → Sum rows or columns

Pattern 3: Finding Constants

Given: f(x) = c·g(x) Find: c → Use ∫f(x)dx = 1

Pattern 4: Probability Ranges

Find: P(a < X < b) → F(b) - F(a) or ∫ₐᵇ f(x)dx

Pattern 5: Independence Check

Check if P(A,B) = P(A)·P(B) Or if P(A|B) = P(A)

🎯 Maximum Likelihood (MLE)

\[\hat{\theta}_{MLE} = \arg\max_\theta L(\theta)\]
What it means: Find parameter that makes observed data most likely.
🎯 Steps:
  1. Write likelihood: L(θ) = P(data|θ)
  2. Take log: ℓ(θ) = log L(θ)
  3. Differentiate: dℓ/dθ
  4. Set to 0 and solve
  5. Verify it's maximum

Common MLEs

  • Binomial p: p̂ = k/n (fraction of successes)
  • Normal μ: μ̂ = x̄ (sample mean)
  • Normal σ²: σ̂² = (1/n)Σ(xᵢ-μ)²
  • Exponential λ: λ̂ = 1/x̄

⚠️ TOP EXAM MISTAKES TO AVOID

❌ MISTAKE 1: Confusing P(A|B) with P(B|A)
✓ FIX: They're different! Use Bayes' Rule to flip.
❌ MISTAKE 2: Thinking mutually exclusive = independent
✓ FIX: If A and B can't both happen, they're dependent!
❌ MISTAKE 3: E[g(X)] = g(E[X])
✓ FIX: Usually NOT true! Must apply g first, then integrate.
❌ MISTAKE 4: Forgetting to normalize
✓ FIX: Always check Σp = 1 or ∫f dx = 1
❌ MISTAKE 5: Using wrong formula for variance
✓ FIX: Var(X) = E[X²] - (E[X])², not E[(X-μ)]²
❌ MISTAKE 6: Ignoring base rates in Bayes
✓ FIX: Prior P(A) matters! Don't ignore it.
