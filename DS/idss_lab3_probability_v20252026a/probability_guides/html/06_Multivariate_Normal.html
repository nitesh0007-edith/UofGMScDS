🎯 Introduction: Beyond One Dimension

So far we've studied random variables one at a time. But real-world data often involves multiple related variables:

  • Height and weight of a person
  • Temperature and humidity
  • Stock prices of different companies
  • Features in machine learning

The multivariate normal distribution is the most important distribution for modeling multiple variables together.

📊 Covariance and Correlation

Covariance

Covariance measures how two variables change together:

\[\text{Cov}(X, Y) = E[(X - E[X])(Y - E[Y])]\]

Interpretation:

  • Positive: When X increases, Y tends to increase
  • Negative: When X increases, Y tends to decrease
  • Zero: No linear relationship

📏 Example: Height and Weight

Person 1: Height = 170cm, Weight = 65kg
Person 2: Height = 180cm, Weight = 75kg
Person 3: Height = 160cm, Weight = 55kg

Observation: Taller people tend to weigh more
→ Positive covariance!

Correlation

Correlation is normalized covariance (always between -1 and 1):

\[\rho(X, Y) = \frac{\text{Cov}(X, Y)}{\sigma_X \sigma_Y}\]

Range: \(-1 \leq \rho \leq 1\)

  • \(\rho = 1\): Perfect positive correlation
  • \(\rho = 0\): No linear correlation
  • \(\rho = -1\): Perfect negative correlation

Covariance Matrix

For multiple variables, we organize covariances in a matrix:

\[\Sigma = \begin{bmatrix} \text{Var}(X_1) & \text{Cov}(X_1, X_2) & \cdots \\ \text{Cov}(X_2, X_1) & \text{Var}(X_2) & \cdots \\ \vdots & \vdots & \ddots \end{bmatrix}\]

Properties:

  • Symmetric: \(\Sigma_{ij} = \Sigma_{ji}\)
  • Diagonal = variances
  • Off-diagonal = covariances

📐 Multivariate Normal Definition

Multivariate Normal Distribution

A random vector \(\mathbf{X} = [X_1, X_2, \ldots, X_d]^T\) follows a multivariate normal distribution:

\[\mathbf{X} \sim \mathcal{N}(\boldsymbol{\mu}, \Sigma)\]

Where:

  • \(\boldsymbol{\mu}\) = mean vector (d-dimensional)
  • \(\Sigma\) = covariance matrix (d × d)

Probability Density Function

\[f(\mathbf{x}) = \frac{1}{(2\pi)^{d/2}|\Sigma|^{1/2}} \exp\left(-\frac{1}{2}(\mathbf{x} - \boldsymbol{\mu})^T \Sigma^{-1} (\mathbf{x} - \boldsymbol{\mu})\right)\]

Don't worry about memorizing this! The key insights:

  • Generalizes 1D normal to multiple dimensions
  • Shape determined by mean \(\boldsymbol{\mu}\) and covariance \(\Sigma\)
  • Contours are ellipsoids

📊 Example: 2D Normal

Mean vector:

\[\boldsymbol{\mu} = \begin{bmatrix} 0 \\ 0 \end{bmatrix}\]

Case 1: Independent variables (diagonal covariance)

\[\Sigma = \begin{bmatrix} 1 & 0 \\ 0 & 1 \end{bmatrix}\]

→ Circular contours, no correlation

Case 2: Positive correlation

\[\Sigma = \begin{bmatrix} 1 & 0.8 \\ 0.8 & 1 \end{bmatrix}\]

→ Elliptical contours tilted diagonally (↗)

Case 3: Negative correlation

\[\Sigma = \begin{bmatrix} 1 & -0.8 \\ -0.8 & 1 \end{bmatrix}\]

→ Elliptical contours tilted opposite (↖)

✅ Properties and Marginals

🔑 Key Property 1: Marginals are Normal

If \(\mathbf{X} \sim \mathcal{N}(\boldsymbol{\mu}, \Sigma)\), then each component is normally distributed:

\[X_i \sim \mathcal{N}(\mu_i, \Sigma_{ii})\]

The marginal of a multivariate normal is normal!

🔑 Key Property 2: Conditionals are Normal

The conditional distribution is also normal (with updated parameters).

If we observe some variables, the conditional distribution of the remaining variables is still normal!

🔑 Key Property 3: Linear Combinations

Any linear combination of normal variables is normal:

\[\mathbf{Y} = A\mathbf{X} + \mathbf{b} \sim \mathcal{N}(A\boldsymbol{\mu} + \mathbf{b}, A\Sigma A^T)\]

This is incredibly useful for transformations!

📐 Example: Sum of Normal Variables

If \(X_1 \sim \mathcal{N}(1, 4)\) and \(X_2 \sim \mathcal{N}(2, 9)\) are independent:

\[Y = X_1 + X_2 \sim \mathcal{N}(1 + 2, 4 + 9) = \mathcal{N}(3, 13)\]

Mean adds, variance adds (for independent variables)!

Independence vs Uncorrelated

⚠️ Important Distinction

Uncorrelated: Covariance = 0

Independent: \(P(X, Y) = P(X)P(Y)\)

Key Facts:

  • Independent → Uncorrelated (always)
  • Uncorrelated → Independent (only for multivariate normal!)

For multivariate normal distributions, uncorrelated = independent!

🌍 Applications

🤖 Application 1: Machine Learning

Gaussian Naive Bayes:

  • Assume features are normally distributed
  • Estimate mean and covariance for each class
  • Use Bayes' rule for classification

Principal Component Analysis (PCA):

  • Find directions of maximum variance
  • Uses eigenvalues/eigenvectors of covariance matrix
  • Dimensionality reduction

📊 Application 2: Portfolio Theory

Stock returns often modeled as multivariate normal:

\[\mathbf{R} \sim \mathcal{N}(\boldsymbol{\mu}, \Sigma)\]

Where:

  • \(\boldsymbol{\mu}\) = expected returns
  • \(\Sigma\) = covariance matrix (captures correlation between stocks)

Portfolio variance: \(\text{Var}(\mathbf{w}^T\mathbf{R}) = \mathbf{w}^T\Sigma\mathbf{w}\)

🎯 Application 3: Kalman Filtering

Tracking objects (e.g., aircraft, submarines):

  • State (position, velocity) ~ Multivariate Normal
  • Observations ~ Multivariate Normal
  • Update beliefs using Bayes' rule
  • All distributions remain normal!

Used in GPS, robotics, autonomous vehicles

🧬 Application 4: Genetics

Multiple traits (height, weight, blood pressure) modeled jointly:

  • Accounts for genetic correlations
  • Identifies genetic variants affecting multiple traits
  • Predicts disease risk

🎓 Summary: Key Takeaways

Multivariate Normal:

\[\mathbf{X} \sim \mathcal{N}(\boldsymbol{\mu}, \Sigma)\]

Key Properties:

  • Marginals are normal
  • Conditionals are normal
  • Linear combinations are normal
  • Uncorrelated = Independent (special to MVN!)

Applications:

  • Machine learning (classification, PCA)
  • Finance (portfolio optimization)
  • Tracking (Kalman filters)
  • Genetics and medicine
