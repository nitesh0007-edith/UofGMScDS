🎯 Introduction: The Power of Randomness

Imagine you want to estimate the value of π (pi). Instead of using calculus, you could:

1. Draw a square with side length 2
2. Inscribe a circle of radius 1 inside it
3. Randomly throw darts at the square
4. Count how many land inside the circle
5. Estimate: π ≈ 4 × (darts in circle) / (total darts)

This is Monte Carlo estimation - using random sampling to solve mathematical problems!

🎲 Random Sampling

Definition: Random Sample

A random sample is a set of values drawn from a probability distribution, where each value is independent and identically distributed (i.i.d.).

\[X_1, X_2, \ldots, X_n \sim P(X)\]

Read as: "\(X_1\) through \(X_n\) are drawn from distribution \(P(X)\)"

🎲 Example 1: Sampling from a Fair Die

Rolling a die 10 times:

Possible outcomes: {1, 2, 3, 4, 5, 6}
Each has probability: 1/6

Sample: [3, 1, 6, 2, 5, 3, 4, 1, 6, 2]

Properties:
- Each roll is independent
- Each follows same distribution (uniform over 1-6)

🎯 Example 2: Sampling from PMF

Submarine location PMF (256 grid squares):

How to sample:

  1. Generate random number \(u\) between 0 and 1
  2. Find which square's cumulative probability contains \(u\)
  3. Return that square as the sample
\[P(\text{Sample} = x_i) = P(X = x_i)\]

🎰 Monte Carlo Estimation

Monte Carlo Method

Use random sampling to estimate quantities that are difficult to compute analytically.

Basic Idea:

\[\text{True Value} \approx \frac{1}{n}\sum_{i=1}^{n} f(X_i)\]

Where \(X_1, \ldots, X_n\) are random samples

Estimating Expected Values

Monte Carlo Expectation

To estimate \(E[f(X)]\):

  1. Draw \(n\) samples: \(X_1, X_2, \ldots, X_n \sim P(X)\)
  2. Compute: \(\hat{\mu} = \frac{1}{n}\sum_{i=1}^{n} f(X_i)\)
\[E[f(X)] \approx \frac{1}{n}\sum_{i=1}^{n} f(X_i)\]

As \(n \to \infty\), the estimate converges to the true value!

🎲 Example: Estimating π

Setup:

  • Unit circle: \(x^2 + y^2 \leq 1\)
  • Enclosing square: \(-1 \leq x, y \leq 1\)
  • Area of circle: \(\pi r^2 = \pi\)
  • Area of square: \(4\)
  • Ratio: \(\pi/4\)

Monte Carlo Procedure:

  1. Sample random points \((x, y)\) uniformly in \([-1, 1] \times [-1, 1]\)
  2. Check if \(x^2 + y^2 \leq 1\) (inside circle)
  3. Estimate: \(\pi \approx 4 \times \frac{\text{points in circle}}{\text{total points}}\)
With 1,000 samples:
Points in circle: 786
π ≈ 4 × (786/1000) = 3.144

With 10,000 samples:
Points in circle: 7,854
π ≈ 4 × (7854/10000) = 3.1416

True value: π = 3.14159...
More samples → Better estimate!

📊 Law of Large Numbers

Law of Large Numbers (LLN)

As sample size increases, the sample average converges to the expected value.

\[\lim_{n \to \infty} \frac{1}{n}\sum_{i=1}^{n} X_i = E[X]\]

In words: With enough samples, the average of your samples will equal the true mean!

💡 Why Monte Carlo Works

The Law of Large Numbers guarantees that Monte Carlo estimates converge to the true value as we take more samples.

Convergence Rate:

\[\text{Error} \propto \frac{1}{\sqrt{n}}\]

To cut error in half, you need 4× more samples!

🎲 Example: Estimating Mean of a Die

True mean:

\[E[X] = \frac{1 + 2 + 3 + 4 + 5 + 6}{6} = 3.5\]

Monte Carlo Estimates:

10 rolls: [4, 2, 6, 1, 3, 5, 4, 2, 6, 3]
Average = 3.6 (error = 0.1)

100 rolls average: 3.52 (error = 0.02)

1,000 rolls average: 3.498 (error = 0.002)

10,000 rolls average: 3.5001 (error ≈ 0)

🌍 Real-World Applications

🎯 Application 1: Estimating Probabilities

Problem: What's the probability of getting exactly 3 heads in 10 coin flips?

Analytical Solution:

\[P(X = 3) = \binom{10}{3} \left(\frac{1}{2}\right)^{10} = \frac{120}{1024} \approx 0.117\]

Monte Carlo Solution:

  1. Simulate 10,000 trials of flipping 10 coins
  2. Count how many trials had exactly 3 heads
  3. Estimate = (count with 3 heads) / 10,000
Trials with exactly 3 heads: 1,173
Estimate: 1,173 / 10,000 = 0.1173
True value: 0.117
Very close!

💰 Application 2: Pricing Options (Finance)

Stock options are priced using Monte Carlo simulation:

  1. Simulate thousands of possible stock price paths
  2. Calculate option payoff for each path
  3. Average the payoffs
  4. Discount to present value

This is the industry standard for complex derivatives!

🎮 Application 3: Game AI

Monte Carlo Tree Search (MCTS):

  1. From current game state, randomly play out to the end
  2. Repeat many times
  3. Choose move that led to most wins

Used in AlphaGo to master Go!

🏥 Application 4: Medical Research

Clinical Trial Simulation:

Before running expensive trials, simulate:

  • Patient outcomes under different treatments
  • Statistical power of the study
  • Required sample sizes

Saves time and resources!

🔍 Application 5: Submarine Search

Estimating Search Effectiveness:

  1. Sample possible submarine locations from posterior
  2. For each location, simulate search outcomes
  3. Estimate probability of finding submarine
  4. Choose search strategy with highest success rate
After 1,000 simulated searches:
- Strategy A: 65% success
- Strategy B: 72% success
→ Choose Strategy B!

Importance Sampling

Advanced: Importance Sampling

Sample from a different distribution to reduce variance:

\[E_P[f(X)] = E_Q\left[f(X) \frac{P(X)}{Q(X)}\right]\]

Why? Sample more from important regions, less from unimportant ones.

Example: Estimating rare event probabilities - sample more from the rare region!

🎓 Summary: Key Takeaways

Random Sampling:

  • Draw independent samples from a distribution
  • Each sample follows the same distribution

Monte Carlo Estimation:

\[E[f(X)] \approx \frac{1}{n}\sum_{i=1}^{n} f(X_i)\]
  • Use random samples to estimate expectations
  • Converges to true value by Law of Large Numbers
  • Error decreases as \(\frac{1}{\sqrt{n}}\)

Applications:

  • Estimating probabilities and integrals
  • Financial modeling and risk analysis
  • Game AI and decision making
  • Scientific simulations
