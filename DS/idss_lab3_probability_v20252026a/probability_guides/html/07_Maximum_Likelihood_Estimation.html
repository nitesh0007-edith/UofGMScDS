🎯 Introduction: Learning from Data

Suppose you flip a coin 10 times and get 7 heads. Question: What's the probability the coin lands heads?

Intuitively: 7/10 = 0.7

Maximum Likelihood Estimation (MLE) formalizes this intuition. It's the most important method for estimating parameters from data!

The Core Idea:

Find parameter values that make the observed data most likely.

If the data is [H, H, T, H, T, H, H, H, H, T]:
- p = 0.7 makes this data more likely than
- p = 0.5 or p = 0.9

📊 Likelihood Function

Likelihood Function

The likelihood is the probability of the data as a function of the parameters:

\[L(\theta) = P(\text{Data} \mid \theta)\]

Key Insight: We treat data as fixed and parameters as variable.

🔑 Likelihood vs Probability

Probability: Data varies, parameters fixed

  • \(P(\text{Data} \mid \theta = 0.5)\) = "How likely is this data if p=0.5?"

Likelihood: Parameters vary, data fixed

  • \(L(p) = P(\text{Data} \mid p)\) = "Which p makes this data most likely?"

🎲 Example: Coin Flips

Data: 3 flips = [H, H, T]

Model: Coin has probability \(p\) of heads

Likelihood:

\[\begin{align} L(p) &= P(H, H, T \mid p)\\ &= P(H \mid p) \times P(H \mid p) \times P(T \mid p)\\ &= p \times p \times (1-p)\\ &= p^2(1-p) \end{align}\]

This is a function of \(p\)! For different values:

p = 0.5: L = 0.5² × 0.5 = 0.125
p = 0.6: L = 0.6² × 0.4 = 0.144
p = 0.67: L = 0.67² × 0.33 = 0.148 ← Maximum!
p = 0.7: L = 0.7² × 0.3 = 0.147
p = 0.9: L = 0.9² × 0.1 = 0.081

Log-Likelihood

Log-Likelihood Function

Taking the logarithm makes calculations easier:

\[\ell(\theta) = \log L(\theta)\]

Why use log?

  • Products become sums: \(\log(ab) = \log a + \log b\)
  • Easier to differentiate
  • More numerically stable
  • Maximizing \(\log L\) = maximizing \(L\) (log is monotonic)

🎯 Maximum Likelihood Estimation

Maximum Likelihood Estimator (MLE)

The MLE is the parameter value that maximizes the likelihood:

\[\hat{\theta}_{MLE} = \arg\max_{\theta} L(\theta) = \arg\max_{\theta} \ell(\theta)\]

In words: Find \(\theta\) that makes the observed data most probable.

Finding the MLE

📐 Standard Approach

  1. Write down the likelihood \(L(\theta)\)
  2. Take the log: \(\ell(\theta) = \log L(\theta)\)
  3. Differentiate: \(\frac{d\ell}{d\theta}\)
  4. Set equal to zero: \(\frac{d\ell}{d\theta} = 0\)
  5. Solve for \(\theta\)
  6. Verify it's a maximum (check second derivative)

📝 Worked Examples

🎲 Example 1: Coin Flips (Complete Derivation)

Data: \(n\) flips with \(k\) heads

Parameter: \(p\) = probability of heads

Step 1: Likelihood

\[L(p) = p^k (1-p)^{n-k}\]

Step 2: Log-likelihood

\[\begin{align} \ell(p) &= \log L(p)\\ &= \log(p^k (1-p)^{n-k})\\ &= k \log p + (n-k) \log(1-p) \end{align}\]

Step 3: Differentiate

\[\frac{d\ell}{dp} = \frac{k}{p} - \frac{n-k}{1-p}\]

Step 4: Set to zero

\[\begin{align} \frac{k}{p} - \frac{n-k}{1-p} &= 0\\ \frac{k}{p} &= \frac{n-k}{1-p}\\ k(1-p) &= p(n-k)\\ k - kp &= np - kp\\ k &= np\\ p &= \frac{k}{n} \end{align}\]

Result:

\[\boxed{\hat{p}_{MLE} = \frac{k}{n}}\]

The MLE is simply the fraction of heads! Matches intuition perfectly.

📊 Example 2: Normal Distribution Mean

Data: \(x_1, x_2, \ldots, x_n\) from \(\mathcal{N}(\mu, \sigma^2)\)

Known: \(\sigma^2\) (variance)

Unknown: \(\mu\) (mean)

Likelihood:

\[L(\mu) = \prod_{i=1}^{n} \frac{1}{\sqrt{2\pi\sigma^2}} \exp\left(-\frac{(x_i - \mu)^2}{2\sigma^2}\right)\]

Log-likelihood:

\[\ell(\mu) = -\frac{n}{2}\log(2\pi\sigma^2) - \frac{1}{2\sigma^2}\sum_{i=1}^{n}(x_i - \mu)^2\]

Differentiate and set to zero:

\[\begin{align} \frac{d\ell}{d\mu} &= \frac{1}{\sigma^2}\sum_{i=1}^{n}(x_i - \mu) = 0\\ \sum_{i=1}^{n} x_i &= n\mu\\ \mu &= \frac{1}{n}\sum_{i=1}^{n} x_i \end{align}\]

Result:

\[\boxed{\hat{\mu}_{MLE} = \bar{x} = \frac{1}{n}\sum_{i=1}^{n} x_i}\]

The MLE is the sample mean!

📐 Example 3: Normal Distribution Variance

Data: \(x_1, \ldots, x_n\) from \(\mathcal{N}(\mu, \sigma^2)\)

Known: \(\mu\)

Unknown: \(\sigma^2\)

Result (after derivation):

\[\boxed{\hat{\sigma}^2_{MLE} = \frac{1}{n}\sum_{i=1}^{n}(x_i - \mu)^2}\]

Note: This is the population variance (divides by n, not n-1)

🎯 Example 4: Exponential Distribution

Model: Waiting times ~ \(\text{Exp}(\lambda)\)

PDF: \(f(x) = \lambda e^{-\lambda x}\)

Data: \(x_1, \ldots, x_n\)

Log-likelihood:

\[\ell(\lambda) = n\log\lambda - \lambda\sum_{i=1}^{n} x_i\]

MLE:

\[\boxed{\hat{\lambda}_{MLE} = \frac{1}{\bar{x}}}\]

Inverse of the sample mean!

✅ Properties of MLEs

🎯 Desirable Properties

1. Consistency:

As \(n \to \infty\), \(\hat{\theta}_{MLE} \to \theta_{\text{true}}\)

With enough data, MLE converges to the truth!

2. Asymptotic Normality:

\[\hat{\theta}_{MLE} \sim \mathcal{N}\left(\theta_{\text{true}}, \frac{1}{nI(\theta)}\right)\]

Distribution becomes normal as \(n\) increases (useful for confidence intervals)

3. Efficiency:

Among unbiased estimators, MLE has the smallest variance (asymptotically)

4. Invariance:

If \(\hat{\theta}\) is the MLE of \(\theta\), then \(g(\hat{\theta})\) is the MLE of \(g(\theta)\)

🔄 Example: Invariance Property

If \(\hat{p} = 0.7\) is the MLE for coin flip probability:

  • MLE for \(p^2\): \(\hat{p}^2 = 0.49\)
  • MLE for \(1-p\): \(1 - \hat{p} = 0.3\)
  • MLE for \(\log p\): \(\log \hat{p} = \log 0.7\)

Just apply the transformation to the MLE!

🎓 Summary: Key Takeaways

Likelihood Function:

\[L(\theta) = P(\text{Data} \mid \theta)\]

MLE:

\[\hat{\theta}_{MLE} = \arg\max_{\theta} L(\theta) = \arg\max_{\theta} \log L(\theta)\]

Standard Procedure:

  1. Write likelihood
  2. Take log
  3. Differentiate
  4. Set to zero and solve

Common MLEs:

  • Binomial \(p\): \(\hat{p} = k/n\)
  • Normal \(\mu\): \(\hat{\mu} = \bar{x}\)
  • Normal \(\sigma^2\): \(\hat{\sigma}^2 = \frac{1}{n}\sum(x_i - \mu)^2\)
  • Exponential \(\lambda\): \(\hat{\lambda} = 1/\bar{x}\)
